LocalStorage Encryption Demo

This page demonstrates the ability to encrypt information stored locally on a client's computer / mobile device.

1. Enter a UserName and Password

Start by setting a UserName and Password. Use any string of numbers or letters which you can remember later.
(UserName and Password are CaSe SensiTive)
Passphrase set!

2. Add Data to LocalStorage

Enter as much information as you like. Each entry will be encrypted and stored in your browser's localStorage.

3. Decrypt Your Data

Enter the same UserName and Password combination to decrypt any data you previously stored. Even if you close your browser and restart your machine, the data will persist and remain encrypted.
